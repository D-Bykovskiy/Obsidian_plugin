# Monitoring Plugin для Obsidian (v1.1.0) 💎

**Monitoring Plugin** — это профессиональный инструмент для руководителей, превращающий Obsidian в высокотехнологичную панель управления (Dashboard). Плагин объединяет почту Outlook, мощь корпоративного ИИ и гибкость Obsidian для автоматизации мониторинга и управления задачами.

---

## 🎨 Дизайн: Aerodynamic Cyber-Glass
Плагин выполнен в ультра-современном стиле с использованием эффектов **Glassmorphism** и **Neon Glow**:
*   **Стеклянные элементы:** Глубокое размытие фона (`blur`) и сатурация создают эффект премиального интерфейса.
*   **Неоновая индикация:** Кнопки с яркими акцентными линиями и "матовым" свечением.
*   **Интерактивность:** Эффект "электрической вспышки" при нажатии и плавная кинетика при наведении.

---

## 📋 Ключевые возможности

1.  **Интеграция с Beeline LLM API** — интеллектуальный анализ инцидентов, создание выжимок и еженедельных аналитических отчетов.
2.  **Единая Панель Управления (Main Dashboard):**
    *   **Дашборд:** Метрики в реальном времени, сводные таблицы проектов и инцидентов.
    *   **Канбан-доска:** Управление задачами и проектами простым перетаскиванием (Drag-and-Drop) с мгновенной синхронизацией кэша.
    *   **Календарь (Timeline):** Визуализация сроков задач и этапов проектов на временной шкале.
3.  **Отслеживание цепочек (Rolling Summary):** Плагин автоматически дополняет существующие заметки новыми подробностями из тредов писем, сохраняя контекст и историю.
4.  **Продвинутая фильтрация:** Настройка умных фильтров по проектам, тегам и ключевым словам для фокусировки на главном.
5.  **Интерактивный AI-Чат:** Встроенный ассистент для помощи в анализе данных и написании ответов прямо в Obsidian.
6.  **Управление внутри заметок:** Встроенный UI-блок в каждой заметке для установки сроков, приоритетов и работы с ресурсами.

## 🛠 Быстрый старт

1.  **Outlook & Python:** Убедитесь, что классический Outlook запущен, а Python 3.10+ установлен.
2.  **Зависимости:** Выполните `pip install pywin32` в терминале.
3.  **Настройка:** Введите API-ключ Beeline в настройках плагина.
4.  **Dashboard.md:** Создайте файл `Dashboard.md` и укажите темы для отслеживания в YAML:
    ```yaml
    tracked_subjects:
      - Авария БД
      - Релиз
    ```

---

## 🚀 Использование
*   **📧 Импорт:** Иконка письма запускает сканирование Outlook и создание заметок.
*   **📊 Дашборд:** Открывается кнопкой на боковой панели (иконка Layout). Переключайтесь между вкладками для разного представления данных.
*   **⚙️ Фильтры:** Используйте кнопку "Редактировать фильтры" на Дашборде для гибкой настройки отображения задач.
*   **Mock-режим:** Включите в настройках `Use Mock Data` для тестирования функций без доступа к корпоративной сети.

---

## ❓ Поддержка
Если карточки переносятся не с первого раза или кнопки не подсвечиваются — убедитесь, что вы используете версию **1.1.0** и перезапустите Obsidian для применения новых стилей.
