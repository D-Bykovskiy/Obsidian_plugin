# Регулярные задачи

## Формат:
# day_name:
#   - "Название задачи" | Время (опционально)
#   - "Ещё задача"

## Пример:
# tue:
#   - "Подготовка еженедельного отчёта" | 14:00
#   - "Отправить status update"

---
